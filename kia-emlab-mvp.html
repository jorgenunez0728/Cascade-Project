COP15 Cascade

Ciclo de vida completo del vehículo: alta con OCR → preacondicionamiento → prueba → liberación → PDF oficial. Firma digital, historial, alertas de vehículos estancados.

67 funciones

Test Plan Manager

Plan anual de producción COP. Análisis por familia, cálculo de requerimiento √N, plan semanal con carry-over, comparación de planes con diff, auto-feed desde Cascade.

45 funciones
🧪

Results Analyzer

Importación batch de ~600 pruebas VETS. Tendencias con líneas de control 2σ/3σ, detección de outliers configurable, búsqueda avanzada por VIN/operador/fecha.

24 funciones
📦

Lab Inventory

Control de 28 cilindros de gas, 3 tambos de combustible, 31 equipos con calibración. Predicción de agotamiento basada en lecturas + pruebas de Cascade. Reporte semanal tipo email.

Evolución

De v1 a v23 en 23 iteraciones

v1 — v12
COP15 Cascade
Digitalización del formato de inspección COP15-F05. OCR para VIN, firma digital, generación PDF, cascade select para 800+ configuraciones vehiculares.
v13
Unificación + Test Plan
3 plataformas en una sola app. Test Plan Manager con análisis de regla COP √N, puntaje de prioridad, plan semanal.
v14 — v18
Results Analyzer + Refinamiento
Importación batch VETS, familias con diff-only, Plan vs Actual, carry-over, config panel, accordion scroll.
v19 — v20
Plan Comparison + Outliers
Diff de planes con historial, nuevos propósitos ND-Emisiones/OBD2, outlier detection 2σ, búsqueda avanzada, alertas de estancamiento.
v21 — v23
Lab Inventory
4° módulo completo: 28 gases + 3 tambos + 31 equipos precargados. Predicción desde Cascade, reporte semanal, timeline de cambios, auto-ID, modal forms.
